# NumGuessr Game

## Coded in Python

A simple console-based Number Guessing game programmed in Python.

### Features:

1. Number generation within User's limit
2. Indication about user's guess at each try
3. Relevant Motivational Quotes when user either wins or loses.
4. Have a slight sarcasm touch :)

### How to test it:

Simply, run the .exe (execution) file for running the program. If it fails, open Command Prompt in that directory and run application using it.

### Why I built this:

1. To refresh my Python Skills
2. To enjoy the game within it's development :)
